# ブラックジャック（コンソール版）

β版完成。  
コンソール上で動きます。  

## 動作確認済み環境

Windows 11 64bit  
Amazon Corretto 21.0.6  
Gradle 8.13  
IntelliJ IDEA 2025.1.2 (Community Edition)  

## 動作方法

依存関係を特に作っていないため、IDEの実行ボタンでも動くはず。  
ダメなら`./gradlew clean build run`でも動きます。  

## 補足

ルール説明やパッケージ構成のリファクタリングなどに無料プランのChatGPTを利用しています。
